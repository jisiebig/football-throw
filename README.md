# football-throw
Its the game. football throw. the one that i kinda made or whatever

Description: Its a game where you throw the football at targets. just kinda click and drag it and try hitting the targets. you get a game over if you can beat a level in 10 balls. Targets progressively get harder to hit as levels progress

To run it localy, simply run index.html on any modern browser and it will automatically launch the game
